# GramSeva – Smart Village Service Platform

## Overview

GramSeva is a Flutter-based mobile application designed to bridge the gap between rural citizens and government services. The application provides a digital platform where villagers can access important information, submit requests, register complaints, and stay updated with government schemes and local announcements.

## Problem Statement

Many rural citizens face difficulties in accessing government services due to lack of awareness, paperwork, and communication gaps. GramSeva aims to simplify these processes through a user-friendly mobile application.

## Features

### Citizen Module

* User Registration and Login
* Profile Management
* View Government Schemes
* Submit Complaints and Requests
* Track Complaint Status
* Receive Notifications and Announcements

### Admin Module

* Manage Citizens
* View and Resolve Complaints
* Publish Village Announcements
* Manage Government Scheme Information
* Monitor User Activities

## Installation

### Prerequisites

* Flutter SDK
* Android Studio
* Firebase Project Configuration

### Steps

1. Clone the repository:

```bash
git clone https://github.com/your-username/gramseva.git
```

2. Navigate to project directory:

```bash
cd gramseva
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the application:

```bash
flutter run
```
